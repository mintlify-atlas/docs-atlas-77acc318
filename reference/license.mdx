## Source material copyright

Deltarune is © Toby Fox. The source game content — characters, story, dialogue, and all original creative work — remains the intellectual property of its creator.

This dataset is a **transcription work**: it documents what is said and shown in the game, processed from video playthroughs. The transcription itself is dedicated to the public domain under CC0. The underlying game content it transcribes is not.

## Fan project precedent

This dataset is consistent with standard fan-project precedent for games like Deltarune and Undertale. Transcript and wiki projects for these games have historically operated without issue under similar terms. The CC0 dedication applies to the transcription work only and does not affect the copyright status of the original game.

If you have questions about what you can do with this dataset, the short answer is: everything. The transcription work has no restrictions.
